Настоящая инструкция предназначена для администратора интернет-магазина «Сибирское здоровье» (индивидуального предпринимателя Гариповой). Инструкция описывает:

- запуск и остановку веб-приложения;
- просмотр поступивших заказов от клиентов;
- добавление, редактирование и удаление товаров в каталоге;
- просмотр сообщений из формы обратной связи;
- резервное копирование данных.

> **Важно:** Для корректной работы сайта необходимо открывать его через локальный веб-сервер (Live Server в VS Code или http-server), а не напрямую из файловой системы (дважды кликнув по index.html). При открытии через протокол `file://` запросы к JSON Server блокируются политикой безопасности браузера.
json-server --watch db.json --port 3000
